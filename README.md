# Curso_Java_coder

Neste curso aprendi  Java do absoluto zero até se tornar um programador profissional, dominando os recursos mais moderno da linguagem. Alguns dos assuntos abordados:

- Algoritmo e Estrutura de Dados

- Fundamentos da Linguagem Java

- Estruturas de Controle

- Classes, Objetos, Métodos

- Orientação a Objeto

- Encapsulamento, Herança, Polimorfismo e Abstração

- Lambdas

- Stream API

- Tratamento de Exceções

- JavaFX

- Banco de Dados Relacional

- Bando de Dados NÃO Relacional (NoSQL)

- JPA (Hibernate)

- Spring Boot
